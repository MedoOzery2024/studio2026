# **App Name**: Mahmoud.AI

## Core Features:

- Homepage Text Display: Display the site name "Mahmoud.AI" and developer name: "Mahmoud Mohamed Mahmoud Abo Elfetouh Ahmed El Ozairy" prominently on the homepage.
- 12-Hour Digital Clock: Display a real-time digital clock showing hours, minutes, and seconds in 12-hour format (AM/PM).
- Date Display: Display the current Gregorian date, Hijri date, and day of the week.

## Style Guidelines:

- Primary color: A deep blue (#3F51B5) to represent knowledge and sophistication.
- Background color: A very light gray (#F0F0F0), nearly white, to provide a clean, uncluttered backdrop.
- Accent color: A muted orange (#FFAB40) to draw attention to interactive elements without overwhelming the user.
- Body and headline font: 'Inter', a sans-serif font known for its readability and modern appearance.
- Use minimalist icons with a consistent stroke width to maintain a clean, modern aesthetic.
- Employ a clean, grid-based layout to ensure a well-organized and intuitive user experience.
- Use subtle transitions and animations to enhance the user experience without being distracting.
